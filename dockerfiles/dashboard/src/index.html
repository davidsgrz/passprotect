PassProtect

Gestion corporativa de credenciales

Plataforma self-hosted en Kubernetes con SSO, 2FA y cifrado de extremo a extremo. Diseñada para equipos que necesitan control total sobre sus secretos.

  • ✓
    SSO con Keycloak + OpenLDAP Autenticacion federada con 2FA obligatorio via TOTP
  • ✓
    Vaultwarden (cifrado AES-256) Boveda compatible con Bitwarden, backups cifrados a PostgreSQL
  • ✓
    WAF ModSecurity + Fail2ban OWASP CRS, rate limiting y ban automatico ante ataques
  • ✓
    Infraestructura hardened Pod Security restricted, NetworkPolicies, TLS 1.3

Auditoria SSL / TLS

Calificacion oficial de Qualys SSL Labs sobre los endpoints HTTPS de la plataforma. Verifica protocolos, cifrados, certificados y vulnerabilidades historicas.

Escaneo de las imagenes Docker propias publicadas en el registry. Detecta CVEs en sistema operativo, librerias del lenguaje y dependencias.

Acerca de este panel

Proyecto

PassProtect — Plataforma corporativa de gestion de contrasenas

Trabajo intermodular ASIR 2025-2026

Autores: Francisco Parra Caparros & David Segura Rodriguez

Herramientas de auditoria

  • Qualys SSL Labs — www.ssllabs.com/ssltest/
  • Trivy — escaneo de CVEs en imagenes Docker (aquasecurity/trivy)
  • Nikto — reconocimiento web (sullo/nikto)
  • SQLMap — pruebas de inyeccion SQL (sqlmap.org)
  • ModSecurity + OWASP CRS — WAF (OWASP CRS)
  • Fail2ban — ban automatico (fail2ban.org)

Infraestructura

VPS: Contabo VPS S (4 vCPU, 8 GB RAM)

Orquestador: MicroK8s 1.30

Registry: Docker Hub (dsegura97)

Namespaces: vaultwarden, auth, monitoring, ingress, cert-manager
